## / 주요 직무 및 역할 /  
* 인프라 설계 및 구축: 서버 증설, 네트워크 구성, 클라우드 환경 도입 등 비즈니스 확장에 필요한 시스템 기반을 마련
* 운영 및 유지보수: 24시간 무중단 서비스를 위해 시스템 모니터링, 장애 진단 및 대응, 보안 패치 등 수행
* 자동화 및 최적화: 클라우드(AWS, Azure 등) 환경에서 인프라를 코드로 관리(IaC)하고, 성능 병목 현상을 해결

## / 필요 핵심 역량 /  

* 📚 필요 핵심 역량기반 지식: Linux/Windows 서버 운영체제(OS), TCP/IP 네트워크, 가상화 기술에 대한 깊은 이해
* 클라우드 및 자동화 도구: AWS, Azure, GCP 등 클라우드 플랫폼과 Docker, Kubernetes 등 컨테이너 환경 활용 능력
* 스크립트 언어: Python, Bash Shell Script 등을 활용한 업무 자동화 및 스크립팅 능력
